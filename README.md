# Frothing
**The No-Nonsense AI Surf Reporter**

Frothing is an AI-powered agent designed to rescue surfers from the "data fatigue" of staring at raw NOAA buoy outputs.

## The Vision
Frothing wraps raw oceanographic data into a human-readable recommendation. 
- **Current Focus:** Station 42098 - Egmont Channel Entrance, FL.

## How it works
Frothing analyzes the relationship between:
Swell Data: Retreived from the NOAA National Data Buoy Center.
Wind & Atmospheric Data: Fetched via the National Weather Service (NWS) API from Venice Municipal Airport (KVNC).
